## 已知问题

- 航战只支持主炮时间轴和战列相关设备

目前所有的舰船只支持一条武器时间轴，这导致航战目前只有主炮时间轴、而没有舰载机时间轴。相应的，第4、5装备槽也只列出了战列相关设备

- 战列公共CD问题 #2

因公共CD的处理逻辑, 战列实际开炮时间在部分情况下会和游戏内不一致.
由于实际机制尚不明确而本人暂时没空进行详细测试, 并且这种情况出现概率较小, 所以暂时不会进行修复.

- 约克城II的'翱翔天际之鹰'技能

由于机制较复杂, 暂时无法模拟
